## 什么是 ShipSaaS？

欢迎使用 ShipSaaS，**一个用于构建 SaaS 应用程序的完整样板**。这个项目旨在通过强大、模块化和可扩展的基础来加速您的开发。您可以按原样使用它，也可以根据自己的需求进行扩展。

人们使用 ShipSaaS 构建 SaaS 应用程序的原因有很多：

- 🚀 **快速且现代** - 基于 Next.js 15 App Router 和 React 19 构建
- 🎨 **精美 UI** - 由 TailwindCSS v4 和 shadcn/ui 组件驱动
- 📱 **响应式** - 在所有设备上都能完美运行
- 🔍 **SEO 优化** - 内置 SEO 功能和元数据管理
- 💳 **支付就绪** - 集成 Stripe，支持多种支付方式
- 🌍 **全球化就绪** - 完整的国际化支持
- 🔐 **默认安全** - 企业级身份验证和安全性

无论您是初学者还是经验丰富的开发者，本文档都将指导您完成 SaaS 应用程序的设置、配置和部署。

本指南假设您对 Next.js 和 Prisma ORM 有一定的了解。如果您对这些不熟悉，请参考[官方文档](#官方文档)，因为我们不会在本指南中涵盖基础知识。

让我们开始吧！

## 技术栈

### Next.js 15
使用 React 19 构建全栈 Web 应用程序的 React 框架。

### Prisma ORM
下一代 ORM，帮助您以类型安全的方式访问数据库。

### TailwindCSS v4
实用优先的 CSS 框架，包含可组合构建任何设计的实用类。

### Shadcn/ui / Magic UI / Animate UI / TailArk
预构建组件和块的集合，您可以使用它们来构建 SaaS 应用程序。

### NextAuth.js
Next.js 应用程序的完整身份验证解决方案，支持多个提供商。

### Resend
现代邮件服务，允许您从应用程序发送邮件。

### Fumadocs
为开发者设计的美观文档框架，灵活且高性能。

### Stripe
SaaS 应用程序最受欢迎和最安全的支付处理器。

### Vercel AI SDK
构建 AI 应用程序的工具集合，开源且免费。

### Next-Intl
Next.js 的国际化库，支持路由、消息和本地化。

### Neon PostgreSQL
自动扩展并提供出色性能的无服务器 PostgreSQL 数据库。

### Umami Analytics
注重隐私的网络分析，在不损害用户隐私的情况下提供洞察。
