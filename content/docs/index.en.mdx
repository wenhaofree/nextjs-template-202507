## What is ShipSaaS?

## Tech Stack

### Next.js 15
The React framework for building full-stack web applications using React 19.

### Prisma ORM
A next-generation ORM that helps you access your database in a type-safe way.

### TailwindCSS v4
A utility-first CSS framework packed with utility classes that can be composed to build any design.

### Shadcn/ui / Magic UI / Animate UI / TailArk
A collection of pre-built components and blocks that you can use to build your SaaS applications.

### NextAuth.js
A complete authentication solution for Next.js applications with support for multiple providers.

### Resend
A modern email service that allows you to send emails from your application.

### Fumadocs
A beautiful documentation framework for developers, flexible and performant.

### Stripe
The most popular and secure payment processor for SaaS applications.

### Vercel AI SDK
A collection of tools for building AI applications, open source and free.

### Next-Intl
Internationalization library for Next.js that supports routing, messages, and localization.

### Neon PostgreSQL
A serverless PostgreSQL database that scales automatically and provides excellent performance.

### Umami Analytics
Privacy-focused web analytics that provides insights without compromising user privacy.
