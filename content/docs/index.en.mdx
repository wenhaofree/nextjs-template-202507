# Introduction

Introducing ShipSaaS, a complete Next.js boilerplate for building modern SaaS applications.

## What is ShipSaaS?

Welcome to ShipSaaS, **a complete boilerplate for building SaaS applications**. This project is designed to accelerate your development with a robust, modular, and extensible foundation. Feel free to use it as is or extend it for your own needs.

People use ShipSaaS to build SaaS applications for a variety of reasons:

- üöÄ **Fast & Modern** - Built with Next.js 15.3.5 App Router and React 19
- üé® **Beautiful UI** - Powered by TailwindCSS v4 and shadcn/ui components
- üì± **Responsive** - Works great on all devices
- üîç **SEO Optimized** - Built-in SEO features and metadata management
- üí≥ **Payment Ready** - Integrated Stripe with multiple payment methods
- üåç **Global Ready** - Complete internationalization support
- üîê **Secure by Default** - Enterprise-grade authentication and security

Whether you're a beginner or an experienced developer, this documentation will guide you through the setup, configuration, and deployment of your SaaS application.

This guide assumes you have some familiarity with Next.js and Prisma ORM. If you don't have any of these, please refer to the [Official Documentation](#official-documentation) as we will not cover the basics in this guide.

Let's get started!

## Scope of This Documentation

This documentation will guide you through configuring, running, and deploying the boilerplate, and will provide links to the official documentation of the underlying technologies where necessary. To fully grasp the boilerplate's capabilities, it's essential to understand these technologies, so be sure to explore their documentation as well.

For anything strictly related to the [ShipSaaS boilerplate](https://shipsaas.net/), this documentation has you covered!

### Additional Technologies

#### React 19.0.0
The library for web and native user interfaces.
- **Official Documentation**: [React Docs](https://react.dev/)
- **Version**: 19.0.0

#### TypeScript 5.x
JavaScript with syntax for types.
- **Official Documentation**: [TypeScript Docs](https://www.typescriptlang.org/docs/)
- **Version**: 5.x

#### Framer Motion 12.23.0
A production-ready motion library for React.
- **Official Documentation**: [Framer Motion Docs](https://www.framer.com/motion/)
- **Version**: 12.23.0

#### Zod 3.23.8
TypeScript-first schema validation with static type inference.
- **Official Documentation**: [Zod Docs](https://zod.dev/)
- **Version**: 3.23.8

Understanding these technologies is crucial for building a successful SaaS application.
